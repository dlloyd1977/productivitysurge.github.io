How to join

  1. Email support@productivitysurge.com with the Apple ID you use for TestFlight (use the button above for a ready-made request).
  2. Accept the TestFlight invitation from Apple when it arrives.
  3. Install the TestFlight app from the Mac App Store if you do not already have it.
  4. Install Markdown A.I. 0.7.4 (2) from TestFlight and launch it.

What to test

1. Clean launch

Confirm the app does not silently open or access ~/Documents/Markdown AI on first launch.

2. Choose Folder

Authorize a folder with the system picker, then create, edit, and save a Markdown file there.

3. Quit & relaunch

Quit completely, relaunch, and confirm access to the authorized folder returns without choosing again.

  • Optionally try Add Root, Remove Root, and Reauthorize Root.
  • Confirm File → Open works for a separately selected file.
  • Check Help → Markdown A.I. Website and Help → Report an Issue / Support if present in your build.

Requirements

  • A Mac with Apple Silicon (this build is arm64)
  • A recent macOS release supported by TestFlight
  • An Apple ID that can accept TestFlight invites

How to report issues

Send feedback to support@productivitysurge.com and include:

  • macOS version and Mac model
  • Expected vs actual behavior
  • Steps to reproduce
  • Screenshots or short screen recordings if useful

Privacy

Editing and preview happen on your Mac. ProductivitySurge does not receive your markdown files. See the Privacy Policy for details.
