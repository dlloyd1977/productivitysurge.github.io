Explorer & Quick Open

Browse indexed folders, restore your last session, and jump to any file with Ctrl+P fuzzy search and incremental match highlighting.

Multi-Document Tabs

Work across notes, specs, and README files without losing context. Local markdown links open directly in new tabs inside the app.

Local Agent (Optional)

Send the current document to a locally running model through Ollama, LM Studio, or any OpenAI-compatible endpoint you configure.

Sandboxed File Access

Markdown AI runs sandboxed and accesses files only through folders and files you explicitly authorize on your Mac.

Session Restore

Reopen tabs, workspace roots, and layout preferences automatically so you can pick up where you left off.
